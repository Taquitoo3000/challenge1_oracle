# Analisis de Tiendas

En este proyecto se tiene una base de datos csv de 4 tiendas pertenecianetes a una empresa.
Se realizo un analisis de:

### Ingresos
Se analizaron los ingresos, tanto totales, netos descontando costos, y obteniendo ganancias netas
### Ventas por categoria
Se analizaron los ingresos de cada tienda por categoria de prodcuto
### Calificaciones
Se calcularon las calificaciones promedio de cada tienda para ver cual tenia mejor aceptacion en los clientes
### Productos
Se analizaron los productos mas vendidos y menos vendidos de cada tienda para analizzar sus fortalezas y en que destacan una sobre otra
### Envios
Se analizaron costos de envio de cada tienda para determinar en donde habia mas gastos de envio
